    合成公式¶

    • 哈迪斯的头盔 + 34以上符文 = 升级符文
    • 33#+ 赦免徽章 +（1-7任意符文）= 获得不同职业的随机技能(1# = 法师,2# = 死灵,3# = 圣骑,4# = 野蛮人,5# = 德鲁伊,6# = 刺客,7# = 亚马逊)
    • 34#+34#=35#
    • 35#+35#=36#
    • 血牛角 + 32# + 巴尔的触手 = 随机橙色盾，带有5-26级随机的信念光环
    • 血牛角 + 32# + 卡戎的眼睛 = 随机橙色甲，带有5-26级随机的信念光环
    • 血牛角 + 32# + 美杜莎的头发 = 随机橙色头，带有5-26级随机的信念光环
    • 血牛角 + 32# + 厄里倪厄斯的羽毛 = 随机橙色手，带有5-26级随机的信念光环
    • 血牛角 + 32# + 塔纳托斯的镰刀 = 随机橙色腰带，带有5-26级随机的信念光环
    • 血牛角 + 32# + 弥诺陶洛斯的角 = 随机橙色鞋，带有5-26级随机的信念光环
    • 血牛角 + 32# + 斯芬克斯的翅膀 = 随机橙色武器，带有5-26级随机的信念光环
    • 血牛角 + 32# + 赫卡忒的魔杖 = 随机橙色项链，带有5-26级随机的信念光环
    • 血牛角 + 32# + 阿刻戎的鳞片 = 随机橙色戒指，带有5-26级随机的信念光环
    • 血牛角 + 32# + 哈迪斯的头盔 = 随机橙色职业专用装备，带有5-26级随机的信念光环、所有技能+2-4
    • 吞噬项链 + 30# + 无孔暗金武器 = 项链获取暗金属性
    • 牛宝 * 2 = 巴尔传送门1次
    • 完美红宝石 + 30# + 大型护身符（未激活的） = 99级 大型护身符（属性随机）
    • 完美红宝石 + 30# + 超大型护身符（未激活的）= 99级 超大型护身符（属性随机）
    • 黄金巨皇冠 + 32# + 完美骷髅 = 99级蓝色巨皇冠（属性随机）
    • 初级飞刀 + 3# = 提升属性
    • 分解毁灭 ： 毁灭 + 体力药剂 = 变为30#
